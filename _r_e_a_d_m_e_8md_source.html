1 Introduction
2 ============
3 
4 The goal of this educational project was to create a distributed (decentralized,
5 peer-to-peer, server independent) chat system for iPad that would work in any
6 Wifi network. The chat uses discovery protocol to locate other users and
7 messages are sent to all clients like in a chat room. Furthermore the
8 transmission of messages is encrypted with RSA small keys.
9

72 Discovery
73 ---------
74 
75 To know how is around in the same network a discovery system was implemented
76 using broadcast UDP packets. In this way it is possible to present a list of
77 currently present clients to the user.
78 
79 Chat
80 ----
81 
82 It sends and receives the messages packing them in a encrypted protocol. All
83 messages are sent to all currently present clients.
84 
85 Encryption
86 ----------
87 
88 It is responsible for generating RSA key pairs for the device, encryption and
89 decryption of messages.
