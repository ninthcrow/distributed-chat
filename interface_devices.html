- (NSMutableArray *) getDevices

Alle Geräteinformationen in einem Array zurückgeben (identifier,username,ip,publicKey) Die Rückgabe devices ist ein Array, in dem für jedes aktive Gerät ein Dictionary mit den jeweiligen Geräteinformationen enthalten ist.

Struktur von devices:

key value example Description
